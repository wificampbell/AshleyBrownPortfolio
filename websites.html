Build A Better Planet

Created during a 3-day-long code-a-thon, this website was developed with the goal of providing a solution to help end pollution. Using Google's Machine Learning, our website is able to detect whether an item can be recycled or not.











Danganronpa Personality Test

I created this personality website where you answer a few questions, and at the end, it tells you which character from the show Danganronpa you are. This website was a good way for me to experiment with variables and JavaScript.










My Activist Toolkit

With this advocacy website, I used my coding skills to raise awareness of a global issue: child abuse. Not only that, but I also added resources that people can call or go to if they need help.
